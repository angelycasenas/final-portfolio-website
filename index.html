About This Project

This website was created as a final assignment to demonstrate my skills in planning, designing, and building a professional website. It follows modern web standards, uses semantic HTML, responsive CSS, and accessibility best practices.

Key Features

Responsive Design

Works smoothly on desktop, tablet, and mobile devices using flexible layouts and media queries.

Accessible Markup

Clean Structure

Well-organized HTML and CSS that is easy to maintain and understand.

AI tools were used to assist in brainstorming layout ideas, refining content, checking code quality, and improving accessibility and responsiveness. This reflects real-world development workflows where developers collaborate with AI to work more efficiently.
